# Project Documentation

This folder contains the current technical documentation for `LiveWallpaperAnimeGlitch`.

## Index

- `ARQUITECTURA_GENERAL.md`: high-level system architecture and ownership by domain.
- `ESTADO_PROYECTO_0_2_0.md`: current status, completed refactors, and pending work.
- `AUDIO_RENDER_Y_SHADERS.md`: audio pipeline, render layers, and effects notes.

## Subfolders

- `plans/`: implementation plans and pending feature roadmaps.
- `status/`: handoff notes and progress snapshots.
- `guides/`: practical guides and improvement notes.
- `archive/`: historical notes kept for reference only.

## Recommended reading order

1. `ESTADO_PROYECTO_0_2_0.md`
2. `ARQUITECTURA_GENERAL.md`
3. `AUDIO_RENDER_Y_SHADERS.md`
4. Relevant files under `plans/` and `status/`

## Maintenance rules

- New active docs should live in `plans/`, `status/`, or `guides/`.
- Root-level docs in `docs/` should be long-lived references only.
- Finished or outdated handoffs should move to `archive/`.
